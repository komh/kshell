                             KShell 시험판
                            ---------------
                            
1. 소개
-------
  
  이 프로그램은 기본적으로 비-한국어판 OS/2 환경의 VIO 프로그램에서 한글 
입출력을 도와주기 위해 만들어졌습니다. 그리고 원리상 재지향(Redirection)과 
같은 성격의 프로그램입니다.

2. 제작 환경
------------
  
  - 한국어판 OS/2 Warp v4 with FixPak #15
  
  - EMX 0.9d fix #4 + gcc 2.8.1
    
3. 실행에 필요한 환경
---------------------

  - EMX 0.9d fix #4 runtime

  - 한글( 또는 해당 언어 )를 표현할 수 있는 글꼴

  - 한글( 또는 해당 언어 ) 입력기. 예) KIME

4. 시험 환경
------------

  한국어판 OS/2 Warp v4 with FixPak #15 와 Warp Server for e-Business with 
Fixpak #3 에서 시험했습니다.

5. 기능
-------

  - VIO 프로그램의 입출력을 재지향(redirection)하여 한글 입출력을 가능하게 
    합니다.

  - 기존의 민초(MINCHO) 글꼴 대신 시스템에서 사용가능한 모든 글꼴을 사용할 수 
    있습니다. 단, 고정폭 글꼴만 가능합니다.

  - 코드 페이지를 따로 지정할 수 있기 때문에 한글 뿐만아니라, 해당 언어 
    입력기와 글꼴만 갖추어진다면 해당 언어의 문자를 입력/출력할 수 있습니다.

6. 설치
-------

  kshell.exe 와 viodmn.exe 를 아무 디렉토리에 같이 넣어두시면 됩니다.
  
7. 사용법
---------

7-1. 메뉴(시스템 메뉴)
---------

  Codepage... : 코드 페이지를 지정합니다.
  Font...     : 글꼴의 종류와 크기를 지정합니다.

8. 제한 사항 또는 알려진 벌레

  - 키보드 입력과 출력만을 지원합니다. 따라서 마우스 기능과 클립보드 기능은 
    지원되지 않습니다.  

  - Ctrl-Break, Ctrl-C, Ctrl-S, Ctrl-P 같은 특수키 조합이 지원되지 않습니다.
  
  - VIO 프로그램의 색상과 KShell 의 색상이 일치하지 않습니다.

  - 배경색이 다를 경우, 잔상이 남을 수 있습니다.
    
  - 폴링(Polling) 기법을 이용하기 때문에, 출력 속도가 떨어지며 CPU 점유율이 
    높습니다.

9. 할 것들...
-------------

  - 폴링대신에 VioRegister 따위를 이용해서 출력 속도를 높인다.
  
  - 마우스와 클립보드를 지원한다.
  
  - VIO 색상을 제대로 지원한다.
  
  - 특수키 조합을 지원한다.

11. 소스 컴파일 하기
--------------------
 
  소스를 컴파일하기 위해서는 EMX 0.9d fix #4 + gcc 2.8.1 과 nmake 가 
필요합니다. nmake 는 ToolKit 에 포함되어 있으며, ToolKit 이 없을 경우, 
http://hobbes.nmsu.edu/pub/os2/dev/util/nmakew.zip 를 이용할 수 있습니다.

  이 모든 것이 준비되었다면, 해당 디렉토리에서 'nmake' 를 실행하시면 됩니다.
  
12. 모듈 설명
-------------

  kshell.exe : VIO 의 내용을 PM 상에 뿌려주고, PM 의 입력을 VIO 에 전해주는 
               프로그램 본체.

  viodmn.exe : 'COMSPEC' 으로 지정되어 있는 명령셸을 실행시키고, VIO 의 
               내용을 kshell.exe 에 전해주고, PM 의 입력을 VIO 입력으로 
               전환시켜주는 프로그램. 단독으로 실행될 수 없습니다.

  test.exe   : VIO 상에서 키가 입력되었을 때 키에 대한 정보를 알려주는 
               프로그램.
